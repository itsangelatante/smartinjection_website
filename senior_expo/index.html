SmartInjection
CSUB Senior Design Expo Project

Integrating Blockchain technology with oil well data to create a secure, uniform, and fast Underground Injection Control Review Process.

View Video Presentation View Presentation Poster

Project Features

Peer-to-Peer Network

A Peer-to-Peer Network allows for each peer in the network to hold an up-to-date version of all oil well data.

Uniform Data Structure

The uniform data structure within the blockchain allows for information of oil well data to be in the same place at different institutions to minimize confusion.

Easy to use UI

Our easy to use user interface allows for oil well data inputs to be stored straight into the blockchain as well as search through the blockchain with a single API number.
